# 🔥 Phoenix Portfolio – Responsive Homepage

A fully responsive personal portfolio homepage built as part of **The Odin Project** curriculum.

This project showcases my web development work through a modern, interactive, and visually styled interface with a custom **phoenix-inspired design system**.

---

## 📚 About The Project

This project is a dynamic portfolio homepage that highlights selected projects and personal information in a clean, responsive layout.

It was built to practice **real-world frontend structure**, focusing on layout systems, responsiveness, and UI design.

Users can:

- View an “About Me” section with personal background
- Browse featured projects
- Access project repositories and live demos
- Interact with subtle animations and hover effects

---

## 🧠 Key Concepts

This project focuses on:

- Responsive design across all screen sizes (320px – 1920px+)
- Layout structuring using Flexbox and CSS Grid
- Component-based rendering using JavaScript
- DOM manipulation without frameworks
- UI layering and positioning (absolute / relative)
- Visual hierarchy and design consistency

---
